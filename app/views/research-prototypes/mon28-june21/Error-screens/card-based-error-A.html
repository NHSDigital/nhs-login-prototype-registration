We do not know which website or app you are trying to access

Reference: CID1004

This can happen if you try to log in directly from a web search engine, bookmark or your browser history.

What you can do

To try again, you should go to the website or app you want to access and select Continue with NHS login.

Make sure you go to the main website or app of the service you want to access. Avoid using bookmarks or links from your browser history.

If that doesn't work, you can see more options to try in our help centre.
